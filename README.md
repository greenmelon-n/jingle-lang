Чтобы начать работу на языке JINGLE нужно создать на рабочем столе отдельную папку, назвав ее JINGLE и закинуть туда файлы jingle.py и программа.jingle
файл программа.jingle будет сама непосредственно содержать код
файл jingle.py содержит интерпретатор созданного языка
После написания кода не забудьте сохранить файл программа.jingle , а затем запустите интерпретатор
